## 🎯 特性

- 💫 透明覆盖窗口
- 🤖 桌面自动化
- 🎨 现代化 GUI 界面
- 🔧 可扩展的插件系统
- 📦 跨平台支持

## 🚀 快速开始

```bash
# 使用Poetry安装
poetry install

# 启动应用
poetry run python -m watchcat
```

## 💡 使用场景

- 桌面自动化测试
- 界面交互录制
- 自动化工作流
- 屏幕监控与分析

## 🛠️ 技术栈

- **PyQt6**: 现代化的 GUI 框架
- **Poetry**: Python 依赖管理
- **OpenCV**: 图像处理
- **NumPy**: 数据处理
- **MSS**: 屏幕捕获
